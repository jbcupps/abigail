# Code of Conduct

## Our Commitment

Abigail is committed to providing a welcoming and inclusive environment for everyone. We expect all participants in our community to treat each other with respect and professionalism.

## Standards

We adopt the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as our code of conduct.

Please read the full text at the link above. The key principles are:

- **Be respectful** -- Treat everyone with dignity and consideration.
- **Be constructive** -- Focus feedback on the work, not the person. Offer suggestions for improvement.
- **Be collaborative** -- Work together toward shared goals. Assume good intent.
- **Be inclusive** -- Welcome people of all backgrounds and experience levels.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, discussions, and any other communication channel associated with the project.

## Reporting

If you experience or witness behavior that does not meet these standards, please report it to the project maintainer via GitHub private messaging or by emailing through the contact methods listed in the repository.

Reports will be reviewed promptly and handled with discretion.
